# News Automation System 🚀

This project automatically:
- Fetches latest news (India)
- Converts it into Hinglish posts using AI
- Prepares content for social media posting

## Tech Stack
- Node.js
- OpenAI API
- News API
- GitHub Actions (for automation)

## Setup
1. Add API keys in .env
2. Run: npm install
3. Run: npm start

## Goal
Daily automated current affairs posting
